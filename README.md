## Description

Compile and install a recent cmake version from source tarball.

## Requirements
### Platform:

* Ubuntu/Debian

## Attributes
### Default recipe attributes:

* `node['cmake']['major']` - cmake's major version (default "2.8")
* `node['cmake']['minor']` - cmake's minor version (default "12.1")
